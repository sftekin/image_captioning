# Image Captioning

## Installing
- copy `captions.csv`, `captins_words.csv`, `eee443_project_dataset_train.h5` and `imid.csv` to *dataset* directory.
- copy all the images to *dataset/images* directory
these files wont be committed.

_Required libraries_:
 - torch
 - scikit-image
 - h5py
 - matplotlib
 - pandas, numpy
